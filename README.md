# 📦 EAM RFID PWA

A Progressive Web Application (PWA) for daily RFID-based asset verification using **Zebra TC21 + RFD4031** integrated with **Hexagon EAM**.

This system ensures assets remain in their designated locations and provides daily inventory validation using RFID scanning.

---

## 🚀 Overview

Staff may relocate assets (e.g., moving chairs from meeting rooms to pantry).

To prevent location mismatch:

- Each asset is tagged with RFID
- Daily scanning is performed using Zebra devices
- The system verifies asset location against Hexagon EAM records
- Mismatches and missing assets are identified instantly
